# 楽曲一覧APIの実装

- 楽曲情報を一覧で取得するAPIを実装しました。
- 検索キーワードによる楽曲名（title, titleKana, titleRoman）のLIKE検索に対応しました。
- ページネーションを実装し、1ページあたりのデフォルト取得件数を20件に設定しました。
- 1＋N問題を回避するため、JPAの `@EntityGraph` および `IN` 句を使用し、関連する情報（オリジナル歌唱メンバー、掲載シングル）を効率的に取得できるようチューニング済みです。
- リポジトリから取得したエンティティはService層でDTOに変換し、Controllerで返却する構成としました。
- 既存の `Member.java` のコンパイルエラー修正も合わせて実施しています。
